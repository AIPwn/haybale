Structs

Config

Various settings which affect how the symbolic execution is performed.

ExecutionManager

An ExecutionManager allows you to symbolically explore executions of a function. Conceptually, it is an Iterator over possible paths through the function. Calling next() on an ExecutionManager explores another possible path, returning a ReturnValue representing the function's symbolic return value at the end of that path.
